## เตรียมก่อนเทส

- [ ] เปิดกะแล้ว
- [ ] มีโต๊ะอย่างน้อย 2 โต๊ะ (A/B)
- [ ] มีสินค้าให้กดอย่างน้อย 2 รายการ

## เคส 1: กดโต๊ะเร็ว ไม่ค้าง

ขั้นตอน:
1. เข้า `นั่งโต๊ะ` -> เปิดโต๊ะ A -> ใส่สินค้า 1 รายการ
2. กด `เลือกโต๊ะ` -> เข้าโต๊ะ B -> กลับโต๊ะ A
3. ทำ A -> B -> A เร็วๆ 3 รอบ

ผลที่ต้องได้:
- [ ] ปุ่มกดติดทุกครั้ง
- [ ] ไม่ค้าง/ไม่หน่วงผิดปกติ
- [ ] ไม่เด้ง error แปลก

## เคส 2: รีโหลดแล้วของในตะกร้าโต๊ะต้องอยู่

ขั้นตอน:
1. อยู่โต๊ะ A แล้วใส่สินค้า 2 รายการ (ยังไม่ชำระ)
2. รีโหลดหน้า 1 ครั้ง
3. เข้า `นั่งโต๊ะ` แล้วเลือกโต๊ะ A เดิม

ผลที่ต้องได้:
- [ ] รายการสินค้าในตะกร้ายังอยู่ครบ
- [ ] จำนวน/ยอดรวมตรงก่อนรีโหลด

## เคส 3: ชำระเสร็จต้องกลับหน้าเลือกโต๊ะ

ขั้นตอน:
1. โต๊ะ A มีสินค้าอย่างน้อย 1 รายการ
2. กด `ชำระเงิน` และยืนยันจนสำเร็จ

ผลที่ต้องได้:
- [ ] ระบบเด้งกลับหน้า `รายการเลือกโต๊ะ` อัตโนมัติ
- [ ] ไม่ค้างอยู่หน้าบิลเดิม
- [ ] ตะกร้าด้านขวาไม่ค้างรายการเก่า
